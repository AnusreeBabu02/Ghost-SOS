# Ghost SOS - Stealth Safety App

In uncomfortable, unsafe, or high-pressure social situations, many individuals ‚Äî particularly women ‚Äî need a subtle and non-confrontational exit strategy.
Most safety apps are obvious and can escalate risk if noticed. There is a need for a stealth-first solution that works instantly without raising suspicion.

**Ghost SOS** is a disguised safety tool designed for individuals in high-pressure or uncomfortable situations. It masquerades as a fully functional calculator, allowing users to discreetly trigger a realistic "Ghost Call" to provide a safe exit strategy.

## ‚ú® Features
* **Stealth Disguise:** A clean, functional Calculator UI that hides the app's true purpose.
* **Silent Trigger:** Tapping anywhere on the calculator starts a 5-second stealth countdown.
* **Dynamic UI Transition:** Automatically switches from a "Calculator" to a realistic "Incoming Call" screen with audio.
* **Ongoing Call Simulation:** Transitions to an "Active Call" screen with a timer to maintain the story under scrutiny.
* **Instant Revert:** A single tap on the "Hang Up" button immediately wipes the call screen and returns to the calculator.

![WhatsApp Image 2026-02-21 at 9 31 21 AM](https://github.com/user-attachments/assets/b841887e-1531-4792-abde-0ac8ab45b073)


1.  **Home:** User opens the Calculator.
2.  **Trigger:** User taps a button; a 5-second timer begins.
3.  **Incoming:** `phone_call.jpeg` displays and `ringtone.mp3` plays.
4.  **Ongoing:** Upon clicking "Accept," the screen switches to `ongoing.jpeg` and `voice.mp3` plays.
5.  **Reset:** Clicking "Hang Up" stops all audio and resets the view to the Calculator.
